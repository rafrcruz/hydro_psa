# Contribuindo com o Hydro PSA

Obrigado por contribuir! Este projeto está em fase inicial.

## Fluxo sugerido

1. Crie uma branch a partir de `work`.
2. Faça mudanças pequenas e focadas.
3. Escreva commits claros (imperativo, objetivo).
4. Abra PR com:
   - contexto;
   - o que mudou;
   - como validar.

## Convenções

- Prefira nomes descritivos para arquivos e funções.
- Evite commits com arquivos temporários ou de ambiente local.
- Quando possível, inclua passos de validação no PR.
